Install
1) composer create-project laravel/laravel ./
2) composer require laravel/breeze --dev
3) php artisan breeze:install (php artisan breeze:install --dark)
4) php artisan migrate
5) npm install
6) npm run dev

Update Or Install from git
1) composer install
2) composer update
3) npm install & npm run dev
